Git 커밋 및 푸쉬를 수행해주세요.

1. `git status`로 변경사항 확인
2. `git diff`로 변경 내용 확인
3. 변경사항이 있으면:
   - 모든 파일 스테이징 (`git add -A`)
   - 변경 내용을 분석하여 적절한 커밋 메시지 작성
   - 커밋 메시지는 한국어로, 간결하게 작성
   - 커밋 실행
   - `git push origin main`으로 푸쉬
4. 결과 요약 출력

커밋 메시지 형식:
- 제목: 변경 내용 요약 (50자 이내)
- Co-Authored-By: Claude Opus 4.5 <noreply@anthropic.com>

$ARGUMENTS 가 있으면 커밋 메시지로 사용하세요.
